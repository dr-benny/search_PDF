# วิธีการใช้งานบน Windows

1. ติดตั้ง Node.js (จำเป็นต้องมี)
   - ดาวน์โหลดได้ที่: https://nodejs.org/
   - ติดตั้งตามปกติ (Next ไปเรื่อยๆ)

2. ติดตั้ง Python (จำเป็นต้องมี)
   - ดาวน์โหลดได้ที่: https://www.python.org/
   - **สำคัญมาก**: ตอนติดตั้ง ให้ติ๊กถูกช่อง "Add Python to PATH" ด้วย

3. การใช้งาน
   - ดับเบิ้ลคลิกไฟล์ `start_app.bat` ได้เลย
   - โปรแกรมจะติดตั้งส่วนเสริมที่จำเป็นให้เองในครั้งแรก
   - จะเปิดหน้าเว็บขึ้นมาให้ใช้งานอัตโนมัติ

# การแก้ปัญหาเบื้องต้น
- ถ้าเปิดแล้วหน้าจอดำ หรือขึ้น Error: ลองคลิกขวาที่ `start_app.bat` แล้วเลือก Run as Administrator
- ถ้าหา Python ไม่เจอ: ลองติดตั้ง Python ใหม่แล้วอย่าลืมติ๊ก "Add Python to PATH"
